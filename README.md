## How To Use
- Clone repository on main or master branch
- run "composer update" and "npm update"
- then copy .env from env.example
- make sure on your .env "url" have port like localhost:8000;
- then run "php artisan migrate --seed"
- run "php artisan key:generate"
- then start the project with run "php artisan serve" and "npm run dev"
